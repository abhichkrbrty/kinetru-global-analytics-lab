## Required Analysis
Each EDA notebook must include:

### 1. Dataset Overview
- Indices included
- Date range
- Frequency

### 2. Price & Returns
- Price series plot
- Daily / weekly returns
- Distribution of returns

### 3. Risk & Drawdown
- Rolling volatility
- Max drawdown
- Drawdown duration

### 4. Cross-Market View (if multiple indices)
- Correlation matrix
- Rolling correlations
